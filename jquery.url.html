jQuery.Url

This class parses/encodes URLs, and provides an interface that parallels the browser’s Location object, except using jQuery style getter/setter methods instead of simple properties.

Note: This class doesn’t support some of the more esoteric features of URLs. If you need this kind of support, you should use a more robust (and heavyweight) implementation such as node.js’s Url module.
